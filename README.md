# Token Supply & Burn Stats

This website is way to track the supply and burn stats of the $TOKEN token.

## Features

- Supply stats
- Burn stats

## Demo Demo - [Link]()

## Install

    1. Clone the repository
       ```bash
         git clone
         ```
    2. Install dependencies
         ```bash
            npm install or yarn install
            ```
    3. Run the app
            ```bash
                npm run dev or yarn dev
                ```

## Tech used

    - Reactjs
    - Vite
    - HTML-CSS
    - Javascript
    - axios (for API calls)
